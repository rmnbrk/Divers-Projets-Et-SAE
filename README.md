# Descriptions des projets et SAÉ

## SAÉ OS
Ce sont des documents explicatifs et techniques pour la configuration d'ordinateurs/d'OS, le choix de composants de PC et l'installation de logiciels sous Linux ou Windows.

## SAÉ exploitation d'une BDD
Création de DashBoard sur Power BI à partir d'une grande base de données afin d'aider une entreprise fictive.

## Projet API Cinéma
DashBoard Power BI automatisé avec des Logic App Microsoft Azure. Les données sont fournies par l'API OMDb.

## Affiche ACE
Poster réalisé dans le cadre du cours de communication afin de présenter ce qu'est un ACE.

## SAÉ MCD MPD SQL
Requêtes SQL et création de Modèle Conceptuel de Données et de Modèle Physique de Données sur PowerAMC pour aider une cause fictive.

## SAÉ installation de services réseaux
Guide dans l'intallation de serveur Nginx et Apache sous Linux.

## Synthèse RSE de l'entreprise Orange
Analyse des pratiques RSE d'Orange dans le cadre du cours d'économie.

## Ergonomie - Plaque à induction
Création et modèlisation d'une plaque à induction ergonomique.

## Maquette Figma SAÉ Lumières
Présentation d'une maquette réalisée sur Figma du sites Lyon Mobilités aménagé pour la fête des Lumières.

## PyTorch Vidéo
